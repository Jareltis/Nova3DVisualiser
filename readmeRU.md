# Nova3DVisualiser

*(In English — [readme.md](readme.md))*

Мультиплеерный **ASCII 3D-рейтрейсер**, рендерящий прямо в терминал, написанный на C# / **.NET 10**. По умолчанию работает на многопоточном CPU-рендерере, с опциональным путём на **GPU** (ILGPU / CUDA) при попиксельном паритете. В комплекте — полноценный встроенный редактор сцены, система миров на JSON, настоящая физика твёрдых тел, PNG-текстуры и TCP-мультиплеер, а весь его UI, включая мастер настройки, работает на собственном консольном рендерере движка, **без внешнего UI-тулкита**.

## Возможности

- **Рендерер CPU или GPU, для каждого мира** — многопоточный CPU-рейтрейсер по умолчанию либо путь ILGPU/CUDA при полном паритете фич (прозрачный откат, если GPU нет).
- **Truecolor ASCII** — 24-битный RGB на ячейку через ANSI, с diff-рендерингом (перезаписываются только изменившиеся ячейки).
- **Система множественного света** — point / directional / spot / area, цветное излучение, вращение, многолучевые прожекторы, конусы круг/квадрат/треугольник, альфа-корректные тени.
- **Две прозрачности** — альфа объекта (сквозная, front-to-back) и блёклость цвета (выцветание к белому).
- **PNG-текстуры + UV** — все примитивы и импортированные меши; тайлинг, выбор грани и фильтрация nearest/bilinear/mipmapped; пиксели передаются пирам.
- **Настоящая физика твёрдых тел** — импульсный решатель: гравитация, пер-объектная коллизия (AABB/OBB), трение, трение качения, упругость, масса, 3D-кувыркание, устойчивые стопки и ходячий персонаж от первого лица.
- **Система миров + встроенный редактор** — каждая сцена — это один `worlds/*.json`; спавн, перемещение, правка и сохранение объектов вживую.
- **Камеры и HUD** — виды от 1-го/2-го/3-го лица, размещаемые Fixed/Follow камеры, разделённый экран и три режима HUD (play / overlay / docked).
- **Мультиплеер** — TCP сервер/клиент с синхронизацией мира (чанкованные меши), синхронизацией физики, живыми правками и чатом.

## Быстрый старт

**Требования:** .NET 10 SDK · Windows · терминал с 24-битным цветом (рекомендуется Windows Terminal) · *(опционально)* GPU NVIDIA/CUDA.

```bash
dotnet build -c Release
dotnet run --project SampleGame -c Release
```

Дальше всё берёт на себя брендированный **мастер настройки** — режим сессии, сетевая роль, мир (создать/загрузить) и сетевые параметры.

## Участие

Nova3DVisualiser пишется исключительно его мейнтейнером и **не принимает вклад в код** — пул-реквесты закрываются без слияния. При этом **отчёты об ошибках, предложения функций и идеи очень приветствуются** через [GitHub Issues](https://github.com/Jareltis/Nova3DVisualiser/issues).
